Monitoring Intensity by Patient Severity

Key Principle: Tailor monitoring intensity to patient risk and clinical status
Higher severity = More frequent monitoring + More active intervention
HIGH INTENSITY

Acute Decompensation

NYHA III-IV, active congestion, "Hospital at Home"

  • Vitals: Twice daily
  • Symptoms: Daily
  • Contact: 24/7 access
  • Response: 6-12 hours
Source: Stevenson et al., JACC 2023 [21]
MODERATE INTENSITY

Active Titration

NYHA II-III, post-discharge, medication optimization

  • Vitals: 5x/week (M-F)
  • Symptoms: Weekly
  • Contact: Weekly → Bi-weekly
  • Response: 24-72 hours
Source: Netherlands Protocol [15], PASSPORT-HF [20]
LOW INTENSITY

Stable / Surveillance

NYHA I-II, on optimal therapy, euvolemic

  • Vitals: Weekly minimum
  • Symptoms: Weekly
  • Contact: Monthly
  • Response: Alert-triggered
Source: Netherlands Protocol [15]

Two-Phase Care Protocols

Unstable Protocol (Intensive)

Enrollment Triggers:

  • • Hospital discharge (congested HF)
  • • Initial HF clinic visit
  • • Ambulatory exacerbation
  • • Newly diagnosed HF
  • • Not yet on optimal therapy

Monitoring Schedule:

Weight, BP, HR5x/week (M-F)
SymptomsWeekly
Telephone consultationsEvery 7-14 days
Goals: Stability post-discharge, achieve optimal therapy, decongestion

Stable Protocol (Maintenance)

Enrollment Criteria:

  • • Optimal therapy for ≥1 month
  • • Euvolemic status
  • • Stable diuretic dose (1 month)
  • • No symptom changes
  • • Adequate self-management skills

Monitoring Schedule:

Weight, BP, HRWeekly minimum
SymptomsWeekly
Face-to-faceReduced frequency
Goal: Prevent deterioration through early detection

Dynamic Protocol Transitions

↑ Escalation (Stable → Unstable)

  • • Symptom worsening
  • • Weight gain (decompensation)
  • • Hospital (re)admission
  • • New atrial fibrillation

↓ De-escalation (Unstable → Stable)

  • • Optimal therapy achieved
  • • Stable weight (1 month)
  • • No diuretic adjustments needed
  • • Euvolemia confirmed
Source: Netherlands Protocol [15]
